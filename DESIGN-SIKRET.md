# סיקרט — מערכת עיצוב (Cyber-Spiritual)

**כיוון:** עתידני · ספיריטואלי · אנונימי · לא “רשת חברתית”  
**רפרנס:** ממשק כהה עם ניאון ציאן/מגנטה/סגול, זוהר (glow), זכוכית (glass), צורות גיאומטריות ואנרגיה רכה.

**עקרון:** המסך מרגיש כמו **חלל מעטפת** — לא דף לבן. פעולות קריטיות (נעילה, פתיחה) מקבלות **אור**.

---

## 1. מילון מותג חזותי

| מושג | ביטוי ויזואלי |
|------|----------------|
| מעטפה | צורה מרכזית — יהלום / משולשים זוהרים / עיגול עם טבעת אור |
| נעילה | חותמת ניאון + אנימציה “נסגר” |
| המתנה לזר | טבעת התקדמות זוהרת, חלקיקים איטיים |
| פתיחה | הבזק עדין מגנטה→ציאן, לא “קונפטי” |
| סוד / רוח | קווים גליים, גраדיאנט רך, לא דמויות דתיות |

## 3. אפקטים (חובה לזהות)

### Glow (זוהר)

- כפתור ראשי: `box-shadow: 0 0 24px rgba(45, 226, 230, 0.45)`
- אייקון פעיל: `filter: drop-shadow(0 0 8px rgba(255, 46, 151, 0.6))`
- **לא** על כל אלמנט — רק CTA, מעטפה, מצב פעיל

### Glassmorphism

```css
.glass-panel {
  background: rgba(18, 26, 53, 0.55);
  backdrop-filter: blur(16px);
  border: 1px solid rgba(45, 226, 230, 0.15);
  border-radius: 20px;
}
```

### צורות רקע (דקורטיבי, 10–15% opacity)

- משולשים low-poly זוהרים (מגנטה/סגול)
- ריבועים צפים איטיים (parallax קל)
- קו אנרגיה גלי (SVG) מאחורי כותרות — “ספיריטואלי”, לא דתי

**הימנעות:** נוף synthwave מלא (הר ושמש) — רק כ**איור נדיר** (onboarding אחד), לא בכל מסך.

## 5. רכיבים (UI Kit)

### כפתור ראשי (CTA)

- מילוי: `--gradient-cta` או `neon.cyan` מלא
- פינות: `border-radius: 14px`
- גובה מינימלי: 52px
- טקסט: `text.onNeon` או לבן
- Glow ציאן
- מצב disabled: `bg.panel`, בלי glow, `text.muted`

**דוגמאות טקסט:** `נעל מעטפה` · `היכנס לתור` · `פתח יחד`

### כפתור משני (outline)

- רקע שקוף, `border: 1px solid rgba(45, 226, 230, 0.5)`
- טקסט `neon.cyan`
- glow עדין בהובר

### כרטיס (כוונה / מצב פתיחה)

- `glass-panel`
- נבחר: מסגרת `gradient-ring` דקה (2px) + glow מגנטה חלש
- אייקון קו דק זוהר (לא emoji)

### שדה טקסט (מעטפה)

- רקע `bg.deep`, מסגרת `rgba(123, 47, 247, 0.3)`
- פוקוס: מסגרת ציאן + glow
- מונה תווים: `text.muted` למטה

### תגית כוונה

- pill: `border-radius: 999px`
- לא נבחר: `bg.panel`
- נבחר: גרדיאנט עדין סגול→מגנטה, טקסט לבן

### מעטפה מרכזית (Hero)

- אלמנט SVG: יהלום ממשולשים / עיגול עם `gradient-ring` מסתובב לאט (8–12s)
- מצב **ממתין:** טבעת מפרצת ציאן
- מצב **נעול:** חותמת סגולה, ללא סיבוב
- מצב **נפתח:** פעימה אחת מגנטה→ציאן

### תמונה (v1.1)

- תצוגה מקדימה בתוך מסגרת זוהרת סגולה
- תג `1 היום` — pill מגנטה קטן על אייקון מצלמה

### Paywall / פרימיום

- כרטיס עם גבול זהב-סגול (`#C9A0FF` → `#7B2FF7`)
- אייקון כתר מינימלי קווי (לא emoji)

## 7. אייקונografia

- סגנון: **קו דק** (1.5–2px), זוהר אופציונלי על active
- נושאים: מעטפה, מנעול, שעון, עין (פתיחה), דגל (דיווח)
- **ללא** emoji בממשק
- **ללא** אייקונים “חברתיים” (לב ורוד, תגובות)

## 8. תנועה (Motion)

| אירוע | משך | סוג |
|--------|------|-----|
| מעבר מסך | 280ms | ease-out, fade + slide קל |
| נעילה | 600ms | scale down + flash מגנטה |
| פתיחה | 800ms | glow expand + fade in תוכן |
| תור | לולאה | סיבוב טבעת 4s linear |
| רקע | איטי | parallax 20–30s |

**נגישות:** `prefers-reduced-motion` — כבה סיבובים, השאר fade בלבד.
